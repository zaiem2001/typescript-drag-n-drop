# Typescript Drag n Drop

##### This is a simple **Drag and Drop** App with `TypeScript` and `HTML`

## To run this into your Machine :

1. git clone Repo URL <br />
2. in the terminal `tsc -W` OR `tsc --watch`. <br/>

- Note: You have to install typescript inorder for this to work.

---

- This app has three branches and according to the Branch name, the code is divided. <br>
- When you run the command `tsc -W`, all the `.js` files will be created in `/dist` and then you can run the app.
